Видеофон со звуком

Эта страница использует пользовательский шрифт Rufont и видео "vessel.mp4" со звуком, которое воспроизводится бесконечно на фоне

Добро пожаловать на демонстрационную страницу! Здесь реализовано несколько ключевых функций:

На фоне воспроизводится видео "vessel.mp4" со звуком в бесконечном цикле. Вы можете управлять звуком видео с помощью элемента управления в правом нижнем углу.

Для текста используется пользовательский шрифт "Rufont", который загружается из файла rufont.ttf. Все текстовые элементы страницы используют этот шрифт.

Все ссылки на странице выделены ярким желтым цветом, что делает их хорошо заметными на фоне видео. При наведении на ссылку вы увидите анимационные эффекты.

Особенности страницы:

  • Видеофон с файлом vessel.mp4 со звуком
  • Пользовательский шрифт Rufont из файла rufont.ttf
  • Желтые ссылки с эффектами при наведении
  • Квадратные элементы управления без закруглений
  • Только управление звуком видео (без музыки)
  • Полностью адаптивный дизайн

Страница полностью адаптивна и корректно отображается на всех устройствах. При изменении размера окна браузера видео и интерфейс автоматически подстраиваются.

Технологии, использованные для создания этой страницы: HTML5, CSS3 с пользовательскими шрифтами и JavaScript для интерактивности.
